# osTicket Help Desk Ticketing System

## Project Overview

This project provides hands-on experience with the osTicket help desk ticketing system. The lab simulates real-world IT support workflows including ticket creation, SLA management, and troubleshooting help desk scenarios.

The environment includes:

- osTicket installation and configuration
- Ticket lifecycle management
- SLA tracking and reporting
- Help Desk ticket troubleshooting

---

## Technologies Used

- osTicket
- MySQL / MariaDB
- Apache / Nginx Web Server
- PHP
- Web Browser (Client Access)
- Remote Administration Tools

---

## Lab Architecture

| Component | Purpose |
|---|---|
| osTicket Server | Web-based help desk system |
| Database Server | Stores ticket data and user accounts |
| Client Workstation | Access tickets, create and update requests |
| Admin Panel | Manage tickets, SLAs, and users |
